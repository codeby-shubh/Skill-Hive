# Skill Hive

**A MERN-stack freelancing marketplace inspired by top freelancing platforms, where users can create gigs, book services, make payments, and leave reviews.**

---

##  Features

- **User Authentication** – Signup / Login system for freelancers and clients
- **Gig Creation & Listing** – Freelancers can create service listings (gigs) with descriptions, pricing, etc.
- **Secure Booking & Payments** – Clients can browse gigs, book them, and make payments (use of online payment gateway)
- **Review System** – Clients can leave ratings and reviews for services received
- **Responsive UI** – Accessible and smooth experience on desktop and mobile devices

---

##  Tech Stack

| Layer       | Technologies                         |
|-------------|--------------------------------------|
| Frontend    | React.js, React Router, Context API  |
| Backend     | Node.js, Express.js, REST API        |
| Database    | MongoDB (with Mongoose ORM)          |
| Authentication | JWT (JSON Web Tokens), bcrypt      |
| Deployment  | (Specify: e.g., Heroku, Vercel, Render, etc.) |
