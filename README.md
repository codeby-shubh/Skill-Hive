# 🐝 Skill Hive – Freelancing Marketplace Platform  

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)  
[![MERN](https://img.shields.io/badge/Stack-MERN-green.svg)]()  
[![Netlify Status](https://api.netlify.com/api/v1/badges/your-id/deploy-status)]()  

Skill Hive ek **freelancing marketplace web app** hai jo freelancers aur clients ko ek hi platform par connect karta hai.  
Freelancers apne **gigs/services** list kar sakte hain, aur clients unhe book karke **securely payments** kar sakte hain.  
Ye app **MERN stack** (MongoDB, Express, React, Node.js) par bana hai aur fully responsive hai.  

---

## ✨ Features  

- 🔐 **User Authentication** – JWT + bcrypt ke saath secure login/signup  
- 🎨 **Modern UI** – React.js aur responsive design  
- 💼 **Gigs Management** – Freelancers apne services list aur manage kar sakte hain  
- 💳 **Payment Gateway Integration** – Secure transactions  
- ⭐ **Ratings & Reviews** – Clients freelancers ko review aur rate kar sakte hain  
- 📊 **Dashboard** – Freelancers aur clients ke liye alag-alag dashboards  
- 📱 **Responsive Design** – Mobile, tablet aur desktop par smooth experience  

---

## 🛠️ Tech Stack  

**Frontend:** React.js, Context API, Axios, TailwindCSS/Bootstrap  
**Backend:** Node.js, Express.js, JWT, bcrypt  
**Database:** MongoDB (Mongoose ORM)  
**Deployment:** Netlify (Frontend), Render/Heroku (Backend)  
**Others:** Payment Gateway API, Cloudinary (for images), GitHub Actions (CI/CD)
